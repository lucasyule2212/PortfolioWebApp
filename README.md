# Portfolio Web App

Personal porfolio project, built in NextJS and deeply inspired in the [Discord](https://discord.com/) application.


## Author

- [@lucasyule2212 - GitHub](https://www.github.com/lucasyule2212)
- [lucasyulerocha - LinkedIn](https://www.linkedin.com/in/lucasyulerocha/)




## Used Techs

**Base:** React & NextJS.

**CMS:** Hygraph.

**State Management:**  Zustand + ContextAPI.

**Styling & Animations:** TailwindCSS, ShadcnUI, RadixUI, CVA (Class Variance Authority), Lottie React, Framer Motion.

**Icons:** React Icons, Lucide Icons.

**Internacionalization:** i18next, react-i18next.

**Rich Text Editor:** TipTap Editor, Emoji Mart (Emoji picker).

**Others:** Immer, React Hook Form, Zod, Boring Avatars (random avatar images), Husky, Commitizen, Commitlint.



## Features

- User onboarding wizard
- Internacionalization (English or Portuguese)
- Static pages generation (by NextJS)
- Rich text messages content + Hygraph (CMS) Integration
- Rich text message input
- External links to:
  - My Github;
  - Project repository at Github;
  - LinkedIn;
  - Mail;
  - Online CV;
- Messages reactions.
- "In development" feature flag



## Learnings...

- Modern, hype and trending CSS with Tailwind.
- Components "Composition Pattern".
- How to deal better with animations.
- Apps Internacionalization.
- Modern state management with Zustand and ContextAPI.
- Integration with modern and powerful CMS like Hygraph.
    - and also a little of GraphQL to build the content queries.
